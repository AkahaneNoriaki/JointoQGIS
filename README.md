# JointoQGIS

JointoQGIS は、Excel / CSV ファイルを QGIS レイヤに自動で Join し、
更新を監視・同期する QGIS プラグインです。

Excel を上書き保存するだけで QGIS の属性テーブルが自動更新されます。

---

## 主な機能

- Excel（.xlsx）→ CSV 自動生成（Excelのロック解除待ち対応）
- CSV 更新監視（30秒 / 60秒間隔）
- OBJECTID による安全な仮想 Join（fid 誤結合防止）
- プロジェクト単位の設定保存・復元（再起動後も設定が維持されます）

---

## 対応環境

- QGIS 3.28 LTR ～ 3.40 LTR
- Windows
- Python 3.11 / 3.12
- 依存ライブラリ: `openpyxl`（QGIS に同梱されていない場合は別途インストール）

---

## インストール方法

1. このリポジトリの「Code」→「Download ZIP」からZIPをダウンロード
2. ZIPを展開し、`JointoQGIS` フォルダを以下のパスに配置

```
C:\Users\<ユーザー名>\AppData\Roaming\QGIS\QGIS3\profiles\default\python\plugins\
```

3. QGIS を起動し、メニューの「プラグイン」→「プラグインの管理とインストール」を開く
4. 「インストール済み」タブで「JointoQGIS」にチェックを入れて有効化

---

## 基本的な使い方

1. メニューの「プラグイン」→「JointoQGIS」→「Excel/CSV 同期」でパネルを開く
2. 「Excel(xlsx)を選択」でExcelファイルを指定
3. 対象シートを選択
4. 「CSVフォルダを選択」でCSV出力先フォルダを指定
5. Join 対象のレイヤと結合キー（通常は OBJECTID）を選択
6. 必要に応じて「設定を保存して次回復元」をON、「自動同期ON」をON
7. Excel を更新・保存すると自動的に QGIS の属性テーブルに反映されます

---

## 注意事項

- Excel ファイルが OneDrive 上にある場合、同期中にロックがかかることがあります。その際はロック解除を待って自動リトライします。
- Join は「仮想 Join」のため、元レイヤのデータは変更されません。
- プロジェクトを保存したファイル（.qgs / .qgz）がある場合のみ設定が保存されます。
